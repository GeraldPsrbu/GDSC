## Installation

1. Clone the repository:

   ```git
   https://github.com/Alvalens/react-tools.git
   ```
2. Navigate to the project directory:

   ```shell
   cd react-tools
   ```
3. Install dependencies:

   ```
   npm install
   ```
4. Start the development server

   ```shell
   npm start
   ```
